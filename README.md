# Análisis de Satisfacción de Productos Skincare — Proyecto Coderhouse

Este repositorio contiene el proyecto final del curso de Data Science, donde se analiza la satisfacción del cliente en productos skincare de Sephora utilizando técnicas de análisis exploratorio, regresión logística y clustering.

## Contenido del proyecto
- Limpieza y tratamiento de datos
- EDA (Exploratory Data Analysis)
- Feature selection
- Modelo supervisado (Clasificación — Regresión Logística)
- Modelo no supervisado (Clustering — K-Means)
- Conclusiones y líneas futuras

## Dataset
Dataset extraído de Kaggle: Sephora Products and Skincare Reviews  
Incluye precios, ratings, reseñas y características de productos.

## Tecnologías utilizadas
- Python
- Pandas
- Seaborn / Matplotlib
- Scikit-learn
- Google Colab
